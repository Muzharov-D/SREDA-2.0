# Авандок.Среда — Pitch Deck Outline
## 40-страничный инвестиционный документ

### sec01 — Титул + Executive Summary + Тезис
- Титульная страница: Авандок.Среда, Цифровой EPAM, Seed-раунд 2026
- Executive Summary: три фазы, три боли, три решения
- Тезис Sequoia: Services — The New Software
- Цифры: $638 млрд рынок ИТ-аутсорсинга, $52.6 млрд рынок ИИ-агентов к 2030
- Почему сейчас: модели готовы, governance отсутствует, опыт не накапливается

### sec02 — Проблема: Три стены
- Стена 1: Агента нельзя просто нанять (нет витрины, собеседования, контракта)
- Стена 2: Агентом нельзя управлять как сотрудником (91% используют, 10% имеют governance)
- Стена 3: Агенты не накапливают опыт (каждый стартует с чистого листа)
- Рыночные данные, цитаты Sequoia, Okta/SailPoint, Cognition/Devin

### sec03 — Решение: Три фазы Среды (overview)
- Метафора фазового перехода: Inside (твёрдое тело) → Talent (жидкость) → Forge (газ)
- Портативная цифровая идентичность: агент помнит контекст при смене фазы
- Единый Data Flywheel питает все три фазы
- Сравнение с EPAM, Accenture, Toptal, Cursor, Devin

### sec04 — Фаза Inside: Нервная система компании
- Пульс компании: нейроны = сотрудники, синапсы = передачи работы
- Люди + цифровые сотрудники в одной команде (52% штата — цифровые)
- RPG-профиль: уровни, квесты, скиллы, мана, достижения
- Гейты и трение как фича: sev1-риск, застрявший импульс
- Сквозные потоки: сделка «Гамма» через все отделы
- Governance: доступы через ИБ, JIT, мгновенное отключение

### sec05 — Фаза Talent: Цифровой аутстаффинг
- Agents Portal: витрина агентов с резюме, рейтингами, компетенциями
- Собеседование с тестовым заданием
- Найм в штат (аутстаффинг) или задача под ключ (аутсорсинг)
- Должностная инструкция, доступы, KPI, биллинг
- Агент работает в каналах компании: Telegram, Slack, email, IDE
- Профиль агента растёт с каждой задачей

### sec06 — Фаза Forge: Цифровая кузница
- Заказ проекта «под ключ»: от задачи до готового артефакта
- Сборка команды агентов под проект: архитектор, разработчик, тестировщик
- Изолированная среда (sandbox, gVisor, Firecracker)
- Human-in-the-loop только на гейтах: архитектура, приёмка
- Асинхронное производство: заказчик апрувит, агенты работают
- Результат: код, документация, развёрнутый сервис, отчёт

### sec07 — Переходы между фазами + пользовательские сценарии
- Лестница доверия: Forge → Talent → Inside
- Обратный путь: Inside → Talent → Forge
- Сценарий 1: CTO пробует Forge для MVP, переходит к Talent для поддержки
- Сценарий 2: CEO начинает с Inside, добавляет Talent, заказывает Forge
- Сценарий 3: COO использует Talent для бэк-офиса, расширяет на Inside
- Портативная идентичность: агент помнит контекст при переходе

### sec08 — Архитектура: Фазовая модульность
- 5 функциональных блоков: Framework, Data Platform, Dashboard, Portal, Channels
- Governance: сквозной слой безопасности
- Как блоки комбинируются под фазу:
  - Inside: все 5 блоков, on-premise или SaaS
  - Talent: Portal + Data Platform + Channels + Dashboard (заказчика)
  - Forge: Portal + Framework + Data Platform + Channels + Dashboard
- LLM Gateway: провайдер-агностичность, fallback, локальные модели
- Sandbox: gVisor, Kata Containers, Firecracker
- Data Plane vs Control Plane

### sec09 — Data Flywheel: Защитный ров
- Выполнил → Усвоил → Стал лучше
- Накопленные навыки нельзя скопировать
- Контекст заказчика — уникальный актив
- Сеть эффектов увеличивает маржу
- Снижение стоимости с ростом моделей
- Сравнение с Hermes Agent (Nous Research)

### sec10 — Бизнес-модель: Три потока выручки
- Поток 1: Оплата за результат (~60% выручки на старте)
- Поток 2: Подписка за штатного агента (~25%)
- Поток 3: Комиссия маркетплейса (~10%)
- Поток 4: Enterprise-подписка за governance (~5%, растущий)
- Экономика: затраты не растут линейно, маржа улучшается
- Сравнение с классическим аутсорсингом

### sec11 — Рынок и конкуренты
- $638 млрд: ИТ-аутсорсинг и сервисы (2025)
- $52.6 млрд: рынок ИИ-агентов к 2030 (CAGR 46%)
- 91% → 10%: используют агентов, но governance есть у 10%
- Конкуренты: Cursor, Devin, GitHub Copilot, AutoGPT, BabyAGI, EPAM, Accenture
- Матрица сравнения: инструмент vs фреймворк vs аутсорсинг vs Среда
- Защитные рвы: данные, governance, портативная идентичность

### sec12 — Дорожная карта

- Фаза 2 (3–6 мес): Skills/MCP, RAG, open-weight модели, Cowork-MVP, metering
- Фаза 3 (6–12 мес): K8s Data Plane, SPIFFE/SPIRE, SIEM, air-gapped, HA
