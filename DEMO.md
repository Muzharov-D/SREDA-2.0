# Сценарий показа · Авандок Среда 2.0 (~7 минут)

> Открыть заранее: прод-ссылку Vercel в полноэкранном Chrome (F11), звук не нужен.
> Запасной вариант: кнопка «Тур по продукту» внизу слева — автопилот по той же истории.
> Хоткей: **Ctrl+K** — поставить задачу Среде из любого места. Клик по логотипу — всегда домой, на Пульс.

---

## 1. Пульс компании (стартовый экран) — 60 сек
**Сказать:** «Это живое сердце компании. В центре — Среда, вокруг — восемь отделов.
Каждая летящая частица — реальная задача: черновик делает цифровой сотрудник, человек проверяет и принимает.
В штате 314: 150 людей и 164 цифровых — больше половины компании уже цифровая».

**Сделать:** нажать «⚡ Запустить рой» — волна задач разлетается по отделам.

## 2. Провал в отдел — 40 сек
**Сделать:** кликнуть узел «Продажи» → открывается пульс отдела.
**Сказать:** «Та же логика на уровень ниже: руководитель видит, чем занят его отдел прямо сейчас».

## 3. Штат отдела — равные сотрудники — 90 сек
**Сделать:** в меню слева «Команда» (или с Пульса → Продажи → Команда).
**Сказать:** «Вот штатное расписание: люди и цифровые сотрудники в одних функциях.
У людей — рой под должность. У цифровых — настоящая должностная инструкция».

**Сделать:** кликнуть цифрового («Скоринг») → показать ДИ справа: обязанности, **границы (что запрещено)**, KPI, эскалация человеку.
**Ключевая фраза:** «Цифровой сотрудник не может отправить КП сам — отправляет человек. Никаких волшебных кнопок».

**Сделать:** в поле «Задача для «Скоринг»…» ввести любую задачу с экрана → Enter → тост + запись в аудит.

## 4. Золотой профиль человека — 60 сек
**Сделать:** у любого человека «профиль →».
**Сказать:** «Профиль собран автоматически из всех систем: 1С, КЭДО, CRM — у каждого поля видно, откуда данные.
Вкладка "Рой и ИИ" — его цифровые подчинённые и постановка задачи рою».

## 5. Канал отдела — люди и цифровые в одном чате — 60 сек
**Сделать:** меню «Канал отдела». Написать в композер: «Подними спящие сделки за Q2» → Enter.
Через секунду цифровой сотрудник отвечает и берёт задачу (запись в аудит).
**Сказать:** «Любой ввод живой — система реагирует. Имя любого участника кликабельно».

## 6. Взгляд CEO — 60 сек
**Сделать:** клик по логотипу (домой) → меню «Дашборд компании».
**Сказать:** «CEO не управляет задачами — он видит, как работа течёт. Вот след сделки "Гамма" через четыре отдела,
вот очередь решений». **Сделать:** нажать «Одобрить» на любом пункте → запись в аудит.

## 7. Управление Средой (владелец платформы) — 60 сек
**Сделать:** переключатель кабинетов справа сверху → «Владелец платформы» → «Бюджеты ИИ».
**Сказать:** «Деньги под контролем: лимит на отдел — жёсткая граница».
**Сделать:** изменить лимит любого отдела (ввести число → Enter) — пересчёт на глазах. Переключить любую политику.
Затем «Штат цифровых сотрудников» — реестр всех 164, поиск, статусы, у каждого руководитель-человек.

## Финал — 20 сек
**Сделать:** клик по логотипу → Пульс.
**Сказать:** «Авандок Среда. Больше, чем результат».

---

## Если что-то пойдёт не так
- Любой экран странный → клик по логотипу (всегда возвращает домой) или F5 (state сбросится — это нормально).
- Вопрос «это уже работает с реальными моделями?» → честно: «Это продуктовый прототип логики и управления;
  слой реальных моделей — следующий этап, архитектура под него спроектирована (маршрутизатор, бюджеты, аудит)».
- Не показывать без подготовки: разделы «Видение» (Суперсила/Путь/Контур) — это питч-материал, не продукт.
